Parcel Delivery System Web Service (PDS WSDL)

The latest WSDL may be found at http://localhost:8080/main/PDS?WSDL


Supported Operations

  • Description

    Returns all users (drivers and customers)

    Parameters

    All parameters must be supplied, but may be null. Parameters that may be null are marked optional.

    • type(string, "driver" or "customer", or "NULL") the type will return all of users with the selected type, if NULL is passed all user types will be returned
    • method(string, "XML" or "JSON", or "NULL") returns either XML or JSON, NULL will default to XML

    Response

    All users returned as method chosen (XML or JSON)

  • Description

    Returns user by id (drivers or customers)

    Parameters

    All parameters must be supplied, but may be null. Parameters that may be null are marked optional.

    • id*(int) user will be returned by its id
    • method(string, "XML" or "JSON", or "NULL") returns either XML or JSON, NULL will default to XML

    Response

    User by id returned as method chosen (XML or JSON)

  • Description

    Returns all parcel by customer

    Parameters

    All parameters must be supplied, but may be null. Parameters that may be null are marked optional.

    • id*(integer) Parcels will be returned by customer id
    • method(string, "XML" or "JSON", or "NULL") returns either XML or JSON, null will default to XML

    Response

    Returns customer parcels as method chosen (XML or JSON)

  • Description

    Returns all parcel by driver

    Parameters

    All parameters must be supplied, but may be null. Parameters that may be null are marked optional.

    • id*(integer) Parcels will be returned by created by id (driver)
    • method(string, "XML" or "JSON", or "NULL") returns either XML or JSON, null will default to XML

    Response

    Returns parcels by created by (driver) as method chosen (XML or JSON)

  • Description

    Returns parcel by id

    Parameters

    All parameters must be supplied, but may be null. Parameters that may be null are marked optional.

    • id*(integer) Parcel will be returned by its id
    • method(string, "XML" or "JSON", or "NULL") returns either XML or JSON, null will default to XML

    Response

    Returns parcel by id as method chosen (XML or JSON)

* Required parameters
